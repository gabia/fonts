## 가비아 마음결체 Typeface_Gabia Maeumgyeol  
마음결체는 사람 사이의 마음을 부드러운 획으로 표현한 가로쓰기 글꼴입니다.
장식성과 가독성을 모두 만족시켜, 제목과 본문에 두루 사용할 수 있습니다.
소설이나 시, 영화 포스터와 같이 감성을 자극하는 매체에 어울리는 글꼴입니다
