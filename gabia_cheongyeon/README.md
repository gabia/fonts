## 가비아 청연체 Typeface_Gabia Cheongyeon  
청연체는 붓으로 써 내려간 손글씨를 현대적으로 재해석한 세로쓰기 글꼴입니다.
시와 같은 느린 호흡의 글에 어울리는 글꼴입니다. 본문용 글꼴이지만, 제목으로도 쓸 수 있습니다.
